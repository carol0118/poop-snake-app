# 便便贪吃蛇

独立小游戏 PWA，和 `word-reader-app` 完全分开。

文件说明：

- `index.html`：页面入口
- `app.js`：游戏逻辑
- `styles.css`：界面样式
- `manifest.webmanifest` / `sw.js`：安装与离线支持
- `server.js`：本地静态服务器

本地运行：

1. 进入 `c:\Users\admin\Desktop\WorkSpace\poop-snake-app`
2. 运行 `npm start`
3. 打开 `http://localhost:4173`
4. 手机浏览器或桌面浏览器里可安装为 APP

GitHub Pages：

- 可直接使用仓库根目录发布
- 说明见 `deploy-github-pages.md`
